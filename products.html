Products

Four product families built on SDR and intelligent RF engineering.

AuraLink Spectrion — Spectrum Analyzer

SDR-based RF spectrum analyzer and monitoring suite. Features scheduled scans, data logging, threshold-based alerts, and a responsive web UI for remote monitoring.

  • Band presets: FM (87.5–108 MHz), VHF Marine (156–162 MHz), ISM 2.4 GHz
  • Live spectrum & waterfall, recording (I/Q & audio)
  • Web-based dashboards; CSV/JSON exports
Demo coming soon

AuraLink Wavionic — Drone/Avionics Receiver

Receiver modules for drone RF detection and avionics bands. Supports ML-based classification pipelines for drone presence detection using SDR front-ends.

  • RTL-SDR/HackRF/USRP front-end support
  • FFT features & spectrogram pipelines
  • On-device inference (optional) and REST APIs

AuraLink Signetics Platform — Signal Processing SDK

Cross-platform SDK for SDR integration, signal processing blocks, and ML inference utilities. Built for GNU Radio and Python/C++ workflows.

  • Feature extraction (PSD, cyclostationary, time–freq)
  • GNU Radio OOT blocks; Python bindings
  • Model serving helpers and dataset tools

AuraLink V2X Suite — Vehicular Communications

Tools and prototypes for NR-V2X Mode 2 and ITS. Includes traffic-aware scheduling experiments, aperiodic message handling, and SUMO-based scenario integration.

  • NR-V2X Mode 2 resource selection experiments
  • Learning-based scheduling (DQN/DDPG/Meta-learning)
  • AoI/SINR modeling, SUMO mobility hooks
